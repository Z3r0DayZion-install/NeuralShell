# GLOBAL_ACCOUNT_AND_REGION_PLANNING

## Goal

Coordinate account/region planning across deployment, compliance, channel, and capacity constraints.

## Core Capabilities

- Account-region matrix with fit overlays.
- Opportunity/risk framing by region.
- Global rollout phasing view.
- Exportable planning pack.

## Scope Guardrails

- No abstract planning disconnected from supported models.
- Outputs remain strategy-usable and concrete.
