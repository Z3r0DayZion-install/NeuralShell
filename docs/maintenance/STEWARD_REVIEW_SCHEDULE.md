# NeuralShell V2.1.8 — Steward Review Schedule

## 1. Routine Windows
Maintenance reviews occur in the following windows (All times Local):
- **Weekly Triage**: Wednesday 09:00 - 11:00.
- **Monthly Health**: First Friday of each month.
- **Quarterly LTS**: End of reach fiscal quarter.

## 2. Ad-hoc Triggers
Escalations from the `STEADY_STATE_ESCALATION_MODEL.md` can trigger an emergency Maintenance Review Board (MRB) meeting within 24 hours.

## 3. Preparation
Stewards must arrive with:
- Updated `ISSUE_REVIEW_BOARD.md`.
- Consolidated support logs from the previous period.
- OMEGA verification result for the current baseline.
