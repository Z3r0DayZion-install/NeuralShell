# PILOT_CONVERSION_KIT

## Objective

Turn pilot results into an evidence-backed expansion recommendation.

## Included Assets

- Pilot kickoff checklist
- 2-week and 4-week review templates
- Proof-of-value worksheet
- Expansion map
- Stakeholder summary template
- Renewal/expansion recommendation template
- Risk and blocker register

## Generator

`npm run pilot:conversion`
