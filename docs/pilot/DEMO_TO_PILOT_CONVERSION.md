# DEMO_TO_PILOT_CONVERSION

## Objective

Create an evidence-based transition from demo completion to pilot launch decision.

## Conversion Outputs

- Demo outcome capture.
- Pilot-fit scorecard by weighted dimensions.
- Required artifact tracker.
- Decision state: `not_fit`, `revisit`, or `ready_for_pilot`.
- Pilot launch pack trigger output.

## Decision Discipline

- Scorecards must include reviewer-attributed evidence.
- Missing required artifacts should block pilot-ready classification.
- Internal blockers must include owner and target date before handoff.
