# Support Horizon Checklist

## Inputs
- GM seal date reference
- Current review date
- Current open maintenance items

## Checklist
- [ ] Months since GM seal calculated and recorded.
- [ ] Current support mode confirmed (Active / Security-Only / EOL transition).
- [ ] Open work items are compliant with current support mode.
- [ ] Non-compliant items are deferred or re-scoped.
- [ ] Communication needs are identified for compatibility or deprecation impact.

## Output Fields
- Review date:
- Months since GM seal:
- Support mode:
- Exceptions:
- Required communications:
- Signoff:
