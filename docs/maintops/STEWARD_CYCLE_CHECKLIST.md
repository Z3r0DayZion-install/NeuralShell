# Steward Cycle Checklist

## Pre-Cycle
- [ ] Confirm baseline remains NeuralShell V2.1.4 Golden Master.
- [ ] Confirm lifecycle stage reference remains V2.1.9 Maintenance Operations Ready.
- [ ] Confirm latest health log row exists.

## Weekly Review
- [ ] Run and capture maintenance gate outputs.
- [ ] Compare repeated signals with the prior two weeks.
- [ ] Classify each signal: observation-only, backlog, or patch-watch.
- [ ] Update backlog state when evidence threshold is met.
- [ ] Update patch-watch state when trigger logic is met.
- [ ] Complete cycle signoff.

## Monthly Review
- [ ] Aggregate weekly trend notes.
- [ ] Run support horizon checklist.
- [ ] Run compatibility and deprecation review.
- [ ] Confirm deferred item re-check dates.
- [ ] Publish monthly summary.

## Exit
- [ ] Signoff captured.
- [ ] Deferred actions have owner and due date.
- [ ] Escalations are assigned.
