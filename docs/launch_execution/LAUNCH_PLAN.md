# NeuralShell Launch Execution: Front 1-4 — Real World Validation

## Front 1 — Controlled Distribution
**Goal:** 5-20 high-signal early users.
- **Distribution:** Private Gumroad link + Direct Outreach.
- **Sourcing:** reddit (r/LocalLLaMA, r/selfhosted, r/osinttools), Indie Hackers, Hacker News.
- **Incentive:** $149 (one-time) for the "Founder's Edition" (v5 + roadmap access).

## Front 2 — User Activation
**Goal:** First meaningful action within 60 seconds of install.
- **The Checklist:**
  1. Open app and run `/sys:health` (Check trust badge).
  2. Drop 3 local `.md` or `.txt` files into the Workbench.
  3. Run `/workbench:scan` to see extraction logic.
  4. Run an AI operator command on the extracted context.

## Front 3 — Feedback Capture
**Goal:** Actionable, structured signal from reality.
- **Channel:** Direct email or a private Discord "Operator Channel".
- **Top Questions:**
  1. "What made no sense in the first 60 seconds?"
  2. "What part of the command syntax felt like a barrier?"
  3. "Did the 'Trust Status' make you feel more or less confident in the app?"

## Front 4 — Iteration Loop
**Goal:** Fast improvements from feedback.
- **Cadence:** 1-Week Sprints.
- **Priority:**
  1. Fix blockers/crashes.
  2. Fix UI/UX confusion.
  3. Polish features based on real usage patterns.
- **Hard Rule:** No new features or Lore expansions during the 4-week rollout.

## Success Criteria (Founder Beta)
- [ ] 5-10 users have installed and completed the first action checklist.
- [ ] At least 1 round of feedback-driven UI polish has been released (v2.1.30).
- [ ] 100% stability (no crashes reported by beta users).
