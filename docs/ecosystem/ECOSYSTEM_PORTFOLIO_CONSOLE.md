# ECOSYSTEM_PORTFOLIO_CONSOLE

## Goal

Operate NeuralShell as a coordinated portfolio of offers, services, deployments, and channels.

## Core Capabilities

- Portfolio map across active offers and service lines.
- Attachment and dependency path visibility.
- Portfolio health status by line.
- Exportable portfolio strategy brief.

## Scope Guardrails

- Portfolio entries map to real supported offers only.
- Attachment paths stay evidence-based and readable.
