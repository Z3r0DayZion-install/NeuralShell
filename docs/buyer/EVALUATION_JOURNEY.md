# EVALUATION_JOURNEY

## Goal

Guide institutional evaluators from first review to pilot request with minimal ambiguity.

## Evaluation Steps

1. Read what the platform is and is not.
2. Review use-case matrix by buyer profile.
3. Run 30-minute walkthrough in demo mode.
4. Generate security/procurement review pack.
5. Record success criteria and pilot scope.

## Fast Path Assets

- Evaluator quickstart
- Security review path summary
- Procurement path summary
- Pilot request template
