# MANAGED_SERVICES_COMMAND_CONSOLE

## Goal

Run multi-account managed-service operations using local import/export data without hidden centralization assumptions.

## Core Capabilities

- Account roster with rollout stage and operator assignment.
- Managed account risk summary by support, drift, and trust posture.
- Cross-account escalation queue.
- Support bundle intake guidance per account.
- Weekly summary export for operations review.

## Scope Guardrails

- No mandatory cloud dependency.
- High-risk accounts are explicit and sortable.
- Summaries remain grounded in imported/local account data.
