# NodeChain Missions

NodeChain Missions extend reactive rules into scheduled operational workflows.

## Included Templates

- nightly fleet health sweep
- post-policy recovery export
- weekly release truth audit
- degraded node incident trigger
- proof-pass relay summary
- update fail freeze + notify

## Mission Controls

- run now
- save schedule
- pause schedule
- dry-run mode
- target selected node set or whole fleet
- history with cancel control

## Guardrails

- dry-run remains first-class.
- mission events are written into runtime event feed.
- incident-trigger missions can open Incident Mode directly.
