# Fleet Control

Fleet Control expands runtime visibility beyond one machine by importing local node bundles.

## Features

- node cards with health, policy, proof, update, relay, and collab posture
- status filters (healthy/degraded/critical/offline)
- two-node compare view for meaningful drift analysis
- per-node event feed drill-in
- local runtime import as a fleet node

## Constraints

- offline-capable by default
- local bundle import only unless explicit relay path is configured
- no fake telemetry injection
