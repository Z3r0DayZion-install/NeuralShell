# Phase 9: MVP Acceptance Gates

Criteria for Phase 9 completion and Alpha sign-off.

## Gate 1: Technical Stability
- **G1.1**: Application launches and connects to the Bridge/LLM in < 5 seconds.
- **G1.2**: Session state persists through the `userData` directory.
- **G1.3**: Zero unhandled IPC errors during standard interaction loop.

## Gate 2: Product Usability
- **G2.1**: A user can attach a workspace and query a file without technical setup (CLI).
- **G2.2**: The UI clearly displays Bridge and LLM connectivity status.

## Gate 3: Demo Readiness
- **G3.1**: Successful execution of the "Golden Flow" (Cold Start -> Prompt -> Success).
- **G3.2**: Installation/Launch path documented for Alpha testers.
