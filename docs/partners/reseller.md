# Reseller Operations

NeuralShell partner operations support local deal registration, margin tracking, co-branded asset packs, and reseller report export.

## Console Capabilities
- Register deals with partner IDs.
- Assign lead ownership and stage.
- Track margin percentage and margin dollars.
- Export partner performance report.
- Generate co-branded partner kit JSON per deal.

## Data Templates
- `partners/ops/deal_registration_template.csv`

## Workflow
1. Open `Ecosystem` -> `Partner Operations`.
2. Import template CSV or JSON.
3. Update ownership, stage, and economics.
4. Export reseller report for monthly review.
