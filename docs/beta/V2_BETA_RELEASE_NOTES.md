# NeuralShell V2.0 Beta: Strategic Sovereignty - Release Notes

## Introduction
NeuralShell V2.0 Beta introduces **Strategic Sovereignty**, our most advanced layer of autonomy and situational awareness. This release moves beyond the reactive "Action -> Result" loop of the Alpha to a "Strategy -> Coordination -> Adaptation" model.

## Core Features

### 🚀 Enhanced Chain Learning
The system now reflects on its own history. Autonomous sequences that have proven reliable in specific workspace environments are now prioritized with higher urgency and clear memory-based rationales.

### 🧬 Contextual Action Evolution
No more one-size-fits-all prompts. Actions now evolve their metadata and instructions to match the technical reality of each workspace—whether it's specialized Electron security scans or conservative Git reviews in low-confidence repositories.

### ⛓️ Cross-Chain Coordination
NeuralShell now understands relationships between your projects. A failure in a backend service now correctly escalates urgency in the linked frontend repository, ensuring that coordination is maintained across the entire development chain.

## What's Next?
This Beta rollout represents the final functional milestone for the V2.0 "Sovereignty" phase. Following user validation, we will transition to stability hardening and performance optimization for the V2.0 Release Candidate.
