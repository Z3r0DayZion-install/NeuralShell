# PROOF_BACKED_FOLLOWUP

## Objective

Generate stage-aware follow-up drafts backed by real artifacts and recent system outputs.

## Supported Follow-Up Types

- Demo follow-up.
- Security-review follow-up.
- Pilot kickoff follow-up.
- Expansion follow-up.
- Renewal follow-up.

## Requirements

- Each claim maps to attached evidence references.
- Missing required evidence is visible before export.
- Include "what changed since last meeting" delta notes.
- Generated output is review-only and not auto-sent.
