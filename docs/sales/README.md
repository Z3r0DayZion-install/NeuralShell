# Enterprise Sales Console

NeuralShell includes a local-first sales workflow so founder/admin teams can track enterprise pipeline state without a cloud CRM dependency.

## What It Tracks
- Lead, discovery, demo, pilot, security review, procurement, deployment, renewal.
- Deal size and owner.
- Next action and next action date.
- Demo/proof/pilot progression and blockers.

## Data Formats
- CSV or JSON import from `sales/lead_pipeline_template.csv`.
- JSON export snapshot from the UI (`Sales Console -> Export`).

## Launch Path
1. Start NeuralShell.
2. Open `Ecosystem` from the top status bar.
3. Choose `Enterprise Sales Console`.
4. Import a CSV/JSON file and manage stage progression.

## Offline Ops
- Data is persisted locally in browser storage.
- No external API or cloud sync is required for baseline operation.
