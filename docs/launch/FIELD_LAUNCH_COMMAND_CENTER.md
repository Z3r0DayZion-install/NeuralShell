# FIELD_LAUNCH_COMMAND_CENTER

## Purpose

Provide one-screen field launch readiness across demo, deployment, training, support, buyer journey, pilot conversion, and commercial packaging.

## Health Inputs

- Demo bundle freshness
- Deployment pack freshness
- Training bundle freshness
- Support playbook freshness
- Buyer asset freshness
- Pilot conversion asset freshness
- Commercial package matrix freshness
- Release truth and strict gate status

## Automation

`npm run field-launch:health`

A failed check indicates missing or stale launch-critical assets.
