# ESCALATION_MATRIX

## Severity Definitions

- SEV1 Critical: active operational outage or trust breach.
- SEV2 High: major degradation or blocked institutional workflow.
- SEV3 Medium: bounded degradation with workaround.
- SEV4 Low: cosmetic or low-impact issue.

## Escalation Ladder

1. L1 Support
2. L2 Support
3. Incident Commander
4. Security Engineering
5. Executive Duty Owner
