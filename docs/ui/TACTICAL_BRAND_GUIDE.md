# NeuralShell V2.1 Brand Guide (Stealth Tactical)

## Visual Identity
The NeuralShell V2.1 identity is built on a "Stealth Tactical" aesthetic: mission-ready, security-first, and visually disciplined.

### Core Palette (Design Tokens)
- **Charcoal (Base)**: `--ns-bg-0: #151816`
- **Dark Forest**: `--ns-bg-1: #1C201D`
- **Tactical Olive (Panel)**: `--ns-panel-0: #354030`
- **Muted Sage**: `--ns-panel-1: #2B3327`
- **Pale Stone (Text)**: `--ns-text-0: #E3E7DC`
- **Muted Stone**: `--ns-text-1: #AEB6A6`
- **Strategic Amber**: `--ns-amber: #D9A441`
- **Priority Red**: `--ns-danger: #B64C3B`
- **Healthy Green**: `--ns-success: #6C9B62`

### Typography
- **Headings & Hero**: **Outfit** (Bold/Heavy Weights)
- **UI & Multi-line Text**: **Inter** (Regular/Semi-Bold)
- **Telemetry & Logs**: **DM Mono**

## Logo Assets
- **Mark (`logo-mark.svg`)**: Shield + 4-node neural network. Used for Electron window icon and favicon.
- **Primary Logo (`logo-primary.svg`)**: Mark + "NeuralShell" Wordmark + "MISSION CONTROL" tagline.
- **Wordmark (`logo-wordmark.svg`)**: Stylized "NeuralShell" text for UI-dense areas.

## Design Principles
1. **State Transparency**: UI must clearly reflect operational states (Running, Blocked, Awaiting Input).
2. **High Contrast**: Legibility under load is prioritized over decorative effects.
3. **Geometric Discipline**: Sharp corners (`--radius-sm: 4px`) and consistent grid-based layout.
