# INSTRUCTOR_GUIDE

## Delivery Model

- Run in demo mode for deterministic classroom outcomes.
- Keep labs tied to actual runtime flows.
- Use exported evidence to assess completion.

## Session Cadence

1. Orientation and safety boundaries.
2. Guided module walkthrough.
3. Hands-on lab execution.
4. Debrief and remediation capture.

## Required Evidence

- Lab output artifact
- Incident/continuity trace where applicable
- Completion acknowledgment
