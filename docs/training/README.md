# Certification & Training

NeuralShell ships local training tracks for:
- Audit Operator
- Deployment Admin
- Compliance Lead
- Marketplace Builder

## Flow
1. Open `Ecosystem` -> `Certification Center`.
2. Complete quiz questions for the selected track.
3. Issue a signed JSON certificate once passing score is met.
4. Verify certificate signatures offline by importing the file.

## Artifacts
- Training track config: `src/renderer/src/config/training_tracks.json`
- Certificate signer utility: `src/renderer/src/utils/signedArtifacts.js`
