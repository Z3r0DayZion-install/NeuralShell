# TRAINING_DELIVERY_PACK

## Purpose

Provide role-based institutional training that works offline and maps directly to runtime behavior.

## Tracks

- Operator
- Security/Compliance
- Support
- Admin/Deployment
- Success/Sales Overview

## Bundle Generation

`npm run training:bundle`

Outputs include track manifests, learner workbooks, scenario lab pack, and exam export metadata.

## Verification

- Training artifacts are generated locally.
- Certificate verification follows signed-artifact trust path.
- Scenario labs use demo seeded states rather than synthetic unsupported behavior.
