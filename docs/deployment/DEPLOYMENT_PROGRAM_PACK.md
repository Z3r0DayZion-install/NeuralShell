# DEPLOYMENT_PROGRAM_PACK

## Scope

Deployment packaging for IT, security, and platform teams across supported operating models.

## Supported Paths

- Standard desktop deployment
- Hardware appliance deployment
- Oversight node deployment
- Air-gapped deployment
- PKI-enabled deployment

## Tooling

- Preflight: `npm run deployment:preflight`
- Post-install: `npm run deployment:postinstall`
- Field readiness check: `npm run field:launch:health`

## Program Controls

- Role-based checklist ownership
- Explicit rollback and decommission runbooks
- Upgrade ring promotion guidance
- Evidence capture after install and during changes

## Do Not Assume

- Cloud connectivity
- Unbounded network egress
- Undocumented support entitlement
