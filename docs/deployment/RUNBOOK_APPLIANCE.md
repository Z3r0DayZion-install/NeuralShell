# RUNBOOK_APPLIANCE

## Appliance Deployment Runbook

1. Select approved appliance profile.
2. Provision first boot with role-aware controls.
3. Validate support diagnostics scope.
4. Verify PKI bindings and update signature state.
5. Register appliance in fleet governance records.

## Retirement

- Export appliance support bundle.
- Revoke trust material.
- Execute secure wipe and retirement checklist.
