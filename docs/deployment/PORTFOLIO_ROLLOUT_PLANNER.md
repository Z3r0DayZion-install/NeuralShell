# PORTFOLIO_ROLLOUT_PLANNER

## Goal

Coordinate deployments across many sites/accounts as a constrained portfolio.

## Core Capabilities

- Portfolio stage matrix by site/account.
- Dependency map and blocker clustering.
- Resource strain indicators.
- Sequencing and rollout calendar support.
- Exportable portfolio summary.

## Scope Guardrails

- Stages align with real NeuralShell deployment paths.
- Bottlenecks are visible without dashboard noise.
- Multi-site and multi-account planning remain first-class.
