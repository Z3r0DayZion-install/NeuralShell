# NeuralShell™ Desktop Bootstrap (Electron)

## Quick Start
```bash
npm install
npm start
```

## Build Installer
```bash
npm run dist
```

## Features
- Loads the existing NeuralShell Phase 700 HTML into a native window
- Secure `preload.js` bridges for file read/write and file picker
- Simple LLM connectivity tester to localhost:11434
- Placeholder console for notes or LLM responses
