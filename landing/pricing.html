Common Questions

Does NeuralShell work without internet?
Yes. With a local model provider like Ollama, NeuralShell runs fully offline. Hardware-bound session recovery, audit export, and all core features work without a network connection.
What does "hardware-bound" mean exactly?
Session encryption keys are derived from your machine's hardware identifiers. A backup encrypted on one machine cannot be decrypted on a different machine. This is verifiable — see the proof page for reproducible test scenarios.
Can I try before buying Pro?
Yes. Download Free, run the hardware-binding proof flow, and verify the claims yourself. Pro unlocks additional operator controls and agent features — but core functionality is fully available in Free.
How does offline activation work for Pro/Enterprise?
After purchase you receive a license file that can be activated without an internet connection. Useful for air-gapped environments and restricted networks.
What OS is supported?
Windows 10 and 11 (x64). The v2.1.29 installer is code-signed. macOS and Linux builds are on the roadmap.
