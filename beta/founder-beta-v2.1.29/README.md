# Founder Beta Package (v2.1.29)

This folder is the lean founder-beta deliverable for external technical testers.

## Included

- `installer/` -> packaged installer
- `docs/` -> tester-facing onboarding, limitations, feedback, proof summary
- `screenshots/` -> packaged self-sell flow screenshots
- `proof/` -> packaged proof report + runtime/packaged parity report

## Excluded On Purpose

- internal audit archives
- broad governance dossiers
- raw release engineering clutter not needed by external testers

## Suggested Tester Order

1. `docs/FOUNDER_BETA_QUICKSTART.md`
2. install from `installer/`
3. run proof flow using `docs/FOUNDER_BETA_PROOF_SUMMARY.md`
4. report friction using `docs/FOUNDER_BETA_FEEDBACK.md`
