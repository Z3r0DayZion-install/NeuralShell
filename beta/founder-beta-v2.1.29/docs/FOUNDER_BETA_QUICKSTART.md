# NeuralShell Founder Beta Quickstart

## What This Is

NeuralShell is a local-first operator shell for running evidence-oriented workflows.
This founder beta is focused on one core outcome: prove trust + safety + ROI in a single flow.

## What You Get

- Windows installer (`NeuralShell Setup 2.1.29.exe`)
- Packaged self-sell proof screenshots
- Packaged proof reports
- This quickstart and supporting beta docs

## Install

1. Run `installer/NeuralShell Setup 2.1.29.exe`.
2. Complete the install wizard.
3. Launch NeuralShell from Start Menu or Desktop shortcut.

## First 5-Minute Test Flow

1. Confirm the top shell loads (`Trust_Node_Sovereign` indicator visible).
2. Click `Run 90s Proof` (or run `/proof`).
3. Click `Run /ROI` in the right rail (or run `/roi`).
4. Create a new workflow from the left rail.
5. Run `/guard`, then click `Save Active`, then `Lock Active`.
6. Unlock the session and verify prior guard output is restored.

## What To Look For

- `/proof` returns the trust/safety/release narrative.
- `/roi` returns the operator-value snapshot.
- Lock/unlock preserves and restores chat state.
- No confusing state loss across the save/lock/unlock cycle.
