THANK YOU FOR INSTALLING NEURALSHELL

NeuralShell is a hardened desktop application for autonomous workflow 
execution and LLM-powered operations.

QUICK START
-----------
1. Launch NeuralShell from your Start Menu or Desktop
2. On first run, configure your LLM provider in Settings
3. Use the Command Palette (Ctrl+Shift+P) to execute workflows

SUPPORTED LLM PROVIDERS
------------------------
• Ollama (local, free)     - Install from ollama.com
• OpenAI GPT-4            - Requires API key
• OpenRouter              - Requires API key  
• Groq                    - Requires API key

OFFLINE CAPABILITY
------------------
NeuralShell is designed to work offline-first. With Ollama installed,
all operations run locally on your machine.

SECURITY
--------
• Code-signed installer (verified by Windows)
• Local-first architecture - your data stays on your device
• Encrypted identity storage
• No telemetry without consent

SYSTEM REQUIREMENTS
-------------------
• Windows 10/11 64-bit
• 4 GB RAM (8 GB recommended)
• 500 MB storage

SUPPORT
-------
• Documentation: https://github.com/Z3r0DayZion-install/NeuralShell
• Issues: https://github.com/Z3r0DayZion-install/NeuralShell/issues

UNINSTALL
---------
Use Windows Settings → Apps → NeuralShell → Uninstall
